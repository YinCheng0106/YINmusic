>指令

|  指令   |  說明  |
|  :----  | :----:  |
|`!play [連結 or 曲名]`|播放|
|`!pause`|暫停|
|`!resume`|恢復播放|
|`!skip`|跳過當前歌曲|
|`!stop`|中斷機器人|
|`!loop <all/one/off>`|循環播放 <列表中的歌曲 / 現在播放歌曲 / 關閉>|
|`!shuffle`|隨機播放|
|`!ping`|機器人狀態|
